はじめに

基本方針

本アプリ「森羅万象アナライザー」（以下「本アプリ」）を開発・運営する開発者（以下「開発者」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本ポリシーは、本アプリにおける情報の取り扱いについて説明するものです。

収集する情報

取得する情報

ユーザーが送信する情報

本アプリは、ユーザーが解析のために選択・撮影した画像を取得します。この画像は解析目的のみに使用され、開発者のサーバーには保存されません。

自動的に取得される情報

以下の情報が第三者サービスによって自動的に収集される場合があります。

  • 広告配信のための識別子・端末情報（Google AdMob）
  • アプリのクラッシュ情報・利用統計（各ストアの標準機能）
  • 課金・購入履歴（RevenueCat、決済処理のみ）

端末内に保存される情報

以下の情報はユーザーの端末内にのみ保存され、外部には送信されません。

  • 解析結果（能力値・解説文・レア度）
  • 解析に使用した画像（アプリ専用フォルダ内）
  • アプリの設定情報（広告除去の購入状態など）
外部サービス

第三者サービスへの情報送信

⚠️ 本アプリは解析機能のため、ユーザーが選択した画像を外部サーバー（OpenAI）に送信します。送信前に必ず本ポリシーをご確認ください。
サービス 送信される情報 目的 プライバシーポリシー
OpenAI 解析対象の画像（圧縮済み） AI による能力値・解説文の生成 OpenAI プライバシーポリシー
Google AdMob 広告識別子・端末情報 広告配信 Google プライバシーポリシー
RevenueCat 購入レシート・ユーザーID 課金処理・購入状態の管理 RevenueCat プライバシーポリシー

OpenAI への画像送信について

OpenAI の API 利用規約に基づき、API 経由で送信されたデータは OpenAI の AI モデルの学習には使用されません。ただし、OpenAI のプライバシーポリシーおよびサービス利用規約が適用されます。詳細は上記リンクをご確認ください。

送信を希望しない場合は、本アプリのご利用をお控えください。

取り扱い方針

情報の利用目的

  • AI による画像解析・能力値の生成
  • 解析結果のローカル保存およびコレクション機能の提供
  • 広告の配信（AdMob）
  • 課金機能の提供（RevenueCat）
  • アプリの改善・不具合の修正
ユーザーへのお願い

画像送信に関する注意事項

以下のような画像の送信はお控えください。

  • 他人の顔が写った画像（本人の同意がない場合）
  • 住所・電話番号・クレジットカード番号など個人情報が写り込んだ画像
  • 未成年者が写った画像
  • 成人向けコンテンツ・暴力的・差別的・違法なコンテンツを含む画像
  • 著作権者の許諾なく著作権のある作品を撮影した画像
  • プライバシーを侵害するおそれのある画像
上記に該当する画像を送信した場合、OpenAI の利用規約違反となる可能性があります。また、本アプリは不適切なコンテンツを検出した場合、解析を拒否します。
お子様の利用

未成年者のプライバシー

本アプリは 13 歳未満のお子様を対象としていません。13 歳未満のお子様から意図的に個人情報を収集することはありません。保護者の方で、お子様が本アプリを通じて情報を送信したとお気づきの場合は、下記連絡先までご連絡ください。

サービス内容

本アプリについて

本アプリは、ユーザーが送信した画像をもとに AI が「ゲーム風の能力値」を生成して表示するエンターテインメントアプリです。

本アプリが表示する能力値・解説文はすべて AI が生成したジョークコンテンツです。実在する人物・動物・物の実際の能力・価値・優劣を示すものではありません。
禁止行為

禁止事項

ユーザーは以下の行為を行ってはなりません。

  1. 他人の同意なく、他人の顔写真や個人が特定できる画像を送信・シェアすること
  2. 成人向けコンテンツ・暴力的・差別的・違法なコンテンツを含む画像を送信すること
  3. 著作権・肖像権・プライバシー権その他の権利を侵害する行為
  4. 特定の個人・団体を誹謗中傷する目的で本アプリの解析結果を使用すること
  5. 本アプリのリバースエンジニアリング・改ざん・不正アクセス
  6. 商業目的での無断利用（開発者の書面による許可がない場合）
  7. その他、法令または公序良俗に違反する行為
知的財産

著作権・知的財産権

本アプリに関する著作権・商標権その他の知的財産権は、開発者に帰属します。

ユーザーが本アプリを通じて生成・保存したカード画像について、ユーザーは個人的・非商業的な範囲で自由に利用・SNS シェアを行うことができます。ただし、シェアする画像に第三者の著作物・肖像が含まれる場合は、ユーザー自身が権利処理の責任を負います。

課金・返金

有料機能について

本アプリには「広告除去」の買い切り購入が含まれます。

  • 購入後の返金は、各ストア（App Store / Google Play）の返金ポリシーに従います
  • 機種変更・再インストール後は「購入を復元」機能でお使いいただけます
  • 開発者都合でのアプリ終了・機能変更が生じた場合の返金は対応しかねます
